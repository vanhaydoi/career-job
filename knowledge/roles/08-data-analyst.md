# Data Analyst — Skill Framework

## Tổng quan
- **Tên role:** Data Analyst
- **Level:** Junior → Mid → Senior → Lead
- **Mô tả:** Phân tích dữ liệu để tìm insight, hỗ trợ ra quyết định kinh doanh. Làm việc với SQL, Python/R, Data Visualization (Tableau/Power BI/Looker), Statistics, A/B Testing, Business Metrics. Kể chuyện bằng dữ liệu (storytelling) và thiết kế dashboard hiệu quả.
- **Tags:** data-analyst, sql, python, tableau, powerbi, looker, statistics, ab-testing, product-analytics, marketing-analytics, data-visualization, business-metrics

## Câu hỏi phỏng vấn mẫu theo kỹ năng

### SQL
#### Basic
- "Viết query lấy top 10 khách hàng theo tổng giá trị đơn hàng trong năm qua, hiển thị: tên khách hàng, email, tổng giá trị, số đơn hàng."
- "Phân biệt WHERE và HAVING? Khi nào phải dùng HAVING?"
- "INNER JOIN vs LEFT JOIN — nếu có khách hàng chưa từng mua hàng, họ có xuất hiện trong kết quả INNER JOIN không? LEFT JOIN thì sao?"
- "Viết query tìm sản phẩm chưa từng được bán trong 6 tháng qua."
- "COUNT(*) vs COUNT(column) vs COUNT(DISTINCT column) khác gì?"

#### Medium
- "LAG() và LEAD() dùng để làm gì? Viết query tính retention rate hàng tháng: % người dùng active tháng T-1 tiếp tục active tháng T."
- "ROW_NUMBER() vs RANK() vs DENSE_RANK() — khi có 2 giá trị bằng nhau, mỗi hàm xử lý thế nào? Viết query lấy top 3 sản phẩm bán chạy mỗi category."
- "Viết query tính running total doanh thu theo ngày và MoM (Month-over-Month) growth rate."
- "CTE vs Subquery vs Temporary Table — khi nào dùng cái nào? Viết query dùng recursive CTE để hiển thị cây phả hệ khách hàng referral."
- "Làm sao tối ưu query bị chậm? Bạn kiểm tra những gì đầu tiên? EXPLAIN plan cho bạn biết gì?"

#### Advanced
- "Thiết kế SQL query tính cohort retention matrix (NxM) cho sản phẩm subscription, với retention được định nghĩa là active ít nhất 1 lần trong tháng."
- "PIVOT/UNPIVOT trong SQL — viết query chuyển dữ liệu monthly revenue từ format hàng ngang (Jan, Feb, Mar columns) sang format hàng dọc (Month, Revenue)."
- "Viết query phát hiện anomaly: tìm ngày có doanh thu thấp bất thường (dưới 3 standard deviation so với trung bình 30 ngày trước đó)."
- "Làm sao thiết kế SQL pipeline cho báo cáo tài chính hàng tháng yêu cầu accuracy 100%? Các bước validation và reconciliation bạn sẽ làm?"

- "Pivot table trong pandas: pivot() vs pivot_table() khác gì?"
- "Vẽ biểu đồ cơ bản với matplotlib: line plot, bar chart, scatter plot — các elements cần có (title, axis labels, legend, grid)."

#### Medium
- "Làm sao merge 3+ DataFrames trong pandas? Các loại join và khi nào dùng merge vs join vs concat?"
- "Apply vs Vectorized operations trong pandas — benchmark thực tế khác biệt thế nào? Tại sao nên tránh iterrows()?"
- "Viết script tự động hóa báo cáo hàng tuần: lấy data từ SQL → xử lý với pandas → tạo chart với matplotlib → export ra PDF/Excel."
- "Plotly vs Matplotlib vs Seaborn — khi nào dùng cái nào cho exploratory analysis vs production dashboard?"
- "Làm sao version control Jupyter Notebook? Các tool hỗ trợ: nbstripout, jupytext, ReviewNB, nbdev?"

#### Advanced
- "Thiết kế A/B testing analysis framework bằng Python: input là experiment metadata + raw events → output là report tự động với statistical test results, confidence intervals, diagnostic checks."
- "Xử lý dữ liệu 50GB với Python mà không dùng Spark — Dask vs Polars vs chunked pandas — so sánh performance và trade-off?"
- "Time Series Analysis với Python: phân tích seasonality, trend, anomaly detection, forecasting cơ bản (ARIMA, Prophet)."
- "Viết custom statistical test (permutation test, bootstrap) thay vì dùng test có sẵn — khi nào cần làm vậy?"

### Data Visualization (Tableau / Power BI / Looker)
#### Basic
- "Tableau: Dimensions vs Measures, Discrete vs Continuous — khác gì? Ảnh hưởng đến chart thế nào?"

- "Dashboard Action trong Tableau: Filter, Highlight, URL — dùng để làm gì? Cho ví dụ."

- "Tableau Calculations: Basic, Table Calculation (RUNNING_SUM, WINDOW_AVG, LOOKUP, TOTAL), LOD Expressions (FIXED, INCLUDE, EXCLUDE) — giải thích và cho ví dụ mỗi loại."
- "Power BI: DAX cơ bản — CALCULATE, FILTER, ALL, SUMX khác SUM thế nào?"
- "Looker: LookML basics — view, dimension, measure, explore, join. Code reusability với extends và refinement."
- "Thiết kế dashboard cho CEO: 1 trang duy nhất, thông tin nào nên có, layout thế nào, refresh frequency?"

- "Thiết kế design system cho dashboard (template, color palette, font, layout grid, chart type decision tree) — triển khai cho team 20+ analysts."
- "LookML: persistent derived tables (PDT), incremental PDT, datagroups — khi nào dùng cái nào? Trade-off giữa query speed vs data freshness?"

### Statistics
#### Basic
- "Phân biệt Mean, Median, Mode? Khi nào Median tốt hơn Mean? Cho ví dụ thực tế."
- "Standard Deviation là gì? Làm sao diễn giải Standard Deviation cho người không chuyên?"
- "Phân biệt Correlation và Causation? Ví dụ về correlation không phải causation (spurious correlation)."
- "Normal Distribution là gì? Central Limit Theorem nói gì và tại sao quan trọng?"
- "Percentile là gì? P50, P90, P99 dùng trong trường hợp nào? Latency P99 vs Average?"

#### Medium
- "Hypothesis Testing: Null Hypothesis, Alternative Hypothesis, p-value, significance level (alpha) — giải thích bằng ngôn ngữ đơn giản."
- "Type I Error (False Positive) vs Type II Error (False Negative) — ví dụ thực tế trong kinh doanh?"
- "t-test: khi nào dùng one-sample, two-sample (independent), paired t-test? Assumptions của t-test?"
- "Confidence Interval là gì? 95% CI = 'có 95% xác suất tham số thật nằm trong khoảng này' — câu này đúng hay sai? Giải thích."
- "Sample Size Calculation: cần bao nhiêu sample để detect 5% lift với 80% power và alpha = 0.05?"

#### Advanced
- "ANOVA vs t-test: khi nào dùng ANOVA? One-way vs Two-way ANOVA?"
- "Chi-square test dùng để làm gì? Kiểm tra independence giữa 2 categorical variables."
- "Regression: Linear vs Logistic — khác biệt về output, assumptions, interpretation. Khi nào dùng LOESS/GAM thay vì linear?"
- "Multicollinearity trong regression là gì? VIF, làm sao phát hiện và xử lý?"
- "Bayesian statistics cơ bản: Prior, Likelihood, Posterior. Bayesian A/B testing khác Frequentist thế nào?"

### A/B Testing
#### Basic
- "A/B Testing là gì? Các bước cơ bản để chạy một A/B test?"
- "Control vs Variant — tại sao cần control group?"
- "Metric trong A/B test: Primary metric, Secondary metric, Guardrail metric — khác gì?"
- "Randomization là gì? Tại sao cần randomize? Nếu không randomize thì sao?"

#### Medium
- "Sample Size cho A/B test: cần biết những gì để tính sample size? MDE, alpha, power, baseline conversion."
- "p-value = 0.03 có nghĩa là gì? Có nên kết luận variant tốt hơn control ngay lập tức không?"
- "Multiple Testing Problem: khi test nhiều metrics hoặc nhiều variants cùng lúc, làm sao điều chỉnh? Bonferroni, FDR, Holm-Bonferroni."
- "Peeking Problem: xem kết quả A/B test hàng ngày và dừng khi p < 0.05 — vấn đề gì xảy ra?"
- "A/A Test là gì? Tại sao nên chạy A/A test? Nếu A/A test fail thì có vấn đề gì?"

#### Advanced
- "Network Effects trong A/B testing: khi dùng LinkedIn/Facebook, user trong control và variant tương tác với nhau → làm sao xử lý? Cluster randomization, ego-network partitioning."
- "Switchback Experiment: là gì? Khi nào dùng? Trade-off so với A/B test truyền thống?"
- "Causal Inference: khi không thể chạy A/B test (ethical, impractical), dùng phương pháp gì? Diff-in-Diff, Synthetic Control, Instrumental Variables, RDD, Propensity Score Matching."
- "Bayesian A/B Testing: prior selection, posterior distribution, probability of being best, expected loss. Lợi thế so với Frequentist?"
- "Thiết kế Experiment Review Process: experiment proposal template, review checklist, ship/no-ship criteria, experiment post-mortem."

- "Thiết kế metric framework cho sản phẩm mới từ đầu: chọn North Star, input/output metrics, counter metrics, metric hierarchy, target setting process."

### Dashboard Design
#### Basic
- "Một dashboard tốt cần những yếu tố gì? Information hierarchy, visual hierarchy, KPI cards, trend indicators."

#### Medium
- "Thiết kế dashboard cho 3 use cases khác nhau: Operational dashboard (real-time, chi tiết), Tactical dashboard (hàng ngày/tuần, action-oriented), Strategic dashboard (hàng tháng/quý, high-level KPI). Khác nhau thế nào?"
- "Dashboard Performance: làm sao dashboard load nhanh khi query data source lớn? Extract vs Live, aggregate tables, cache."
- "Mobile dashboard design: constraint và best practice khác desktop thế nào?"
- "Dashboard lifecycle management: từ request → prototype → feedback → publish → maintain → deprecate."

#### Advanced
- "Thiết kế self-service dashboard framework: template chuẩn, data source certification, metric glossary integration, dashboard quality score."
- "Dashboard ROI measurement: làm sao biết dashboard có đang được dùng không? Adoption rate, view count, time spent, decisions influenced."
- "Embedded Analytics: nhúng dashboard vào sản phẩm — technical considerations (SSO, row-level security, white-label, performance), product considerations (user experience, customization)."
